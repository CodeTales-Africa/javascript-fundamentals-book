# Chapter 1 Exercises

Welcome to the Chapter 1 exercises for **JavaScript Fundamentals** by **CodeTales Africa**.

These exercises are designed to reinforce the concepts introduced in **Chapter 1 – Introduction to JavaScript**.

---

## Learning Goals

By completing these exercises, you will strengthen your understanding of:

- What JavaScript is
- The purpose of programming languages
- The relationship between HTML, CSS, and JavaScript
- JavaScript runtime environments
- JavaScript engines
- Browser Developer Tools
- Writing your first JavaScript program
