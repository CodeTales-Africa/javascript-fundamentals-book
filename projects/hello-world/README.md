# Hello World

This project accompanies **Chapter 1 – Introduction to JavaScript** in the **JavaScript Fundamentals** book by **CodeTales Africa**.

---

## Project Overview

The objective of this project is to introduce beginners to writing and executing their first JavaScript program.

By completing this project, you will learn how to:

- Create a basic HTML document.
- Link an external JavaScript file.
- Execute JavaScript in a web browser.
- Display output using `console.log()`.
- Display messages using `alert()`.
- Use the browser's Developer Tools.

## How to Run the Project

1. Open the `hello-world` folder.

2. Double-click `index.html`

OR

Open the folder in **Visual Studio Code** and use **Live Server**.

3. Open your browser.

4. Press **F12**.

5. Click the **Console** tab.

6. Observe the output generated by the JavaScript program.

---

## Expected Console Output


## Learning Outcomes

After completing this project, you should be able to:

- Understand how JavaScript files are connected to HTML.
- Execute JavaScript in a web browser.
- View output in the Developer Console.
- Write simple JavaScript statements.
- Recognize the basic workflow of a JavaScript program.
