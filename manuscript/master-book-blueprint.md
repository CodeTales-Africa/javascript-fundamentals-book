# Vision

CodeTales JavaScript Fundamentals is designed to become one of the most practical, beginner-friendly JavaScript books available.

The goal is to take readers from having no JavaScript knowledge to confidently building real-world applications while understanding not only *how* JavaScript works but also *why* it works.

Rather than focusing on memorization, this book emphasizes understanding, experimentation, and hands-on coding.

# Teaching Philosophy

Every concept follows this learning sequence:

1. Explain the concept
2. Show a simple example
3. Explain the code line by line
4. Show common mistakes
5. Show best practices
6. Apply it to a real-world scenario
7. Reinforce learning with exercises
8. Provide complete solutions
9. Extend learning with a mini project

This approach ensures that readers build understanding before memorizing syntax.

# Code Style

All code examples should:

- Use meaningful variable names
- Follow modern JavaScript standards
- Prefer const over let when values do not change
- Avoid var except for historical explanation
- Include explanatory comments only when they add value
- Be consistently formatted using Prettier
- Favor readability over cleverness

---

# Example Quality Standards

Every example should:

- Compile correctly
- Run correctly
- Demonstrate one primary concept
- Include expected output
- Explain why the code works
- Mention common mistakes where relevant
